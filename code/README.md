# Code

このフォルダには、各数学トピックのPython実装例が含まれています。

## フォルダ構成

- `linear_algebra/`: 線形代数の実装例（ベクトル演算、行列計算など）
- `calculus/`: 微分積分の実装例（数値微分、数値積分など）
- `probability_statistics/`: 確率統計の実装例（分布、統計量など）
- `optimization/`: 最適化アルゴリズムの実装例（勾配降下法など）

## 使用方法

各Pythonファイルは独立して実行できます。必要なライブラリをインストールしてください。

```bash
pip install numpy scipy matplotlib
```
