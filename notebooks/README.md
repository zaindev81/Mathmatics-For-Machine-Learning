# Notebooks

このフォルダには、各数学トピックを実装・可視化したJupyterノートブックが含まれています。

## フォルダ構成

- `01_linear_algebra/`: 線形代数の実装と可視化
- `02_calculus/`: 微分積分の実装と可視化
- `03_probability_statistics/`: 確率統計の実装と可視化
- `04_optimization/`: 最適化アルゴリズムの実装

## 使用方法

各ノートブックは独立して実行できます。必要なライブラリをインストールしてから実行してください。

```bash
pip install numpy matplotlib scipy jupyter
```
