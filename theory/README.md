# Theory

このフォルダには、機械学習に必要な数学の理論的説明が含まれています。

## フォルダ構成

- `linear_algebra/`: 線形代数の理論
- `calculus/`: 微分積分の理論
- `probability_statistics/`: 確率統計の理論
- `optimization/`: 最適化理論
- `basics_japanese.md`: 基礎数学（日本語）

## 学習の進め方

1. まず `basics_japanese.md` で基礎を確認
2. 各トピックのフォルダで詳細な理論を学習
3. 対応するノートブックやコードで実装を確認
